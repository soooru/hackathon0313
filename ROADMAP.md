# 출근장터 로드맵

---

## v1.0 — 완료 ✅

- 이메일 인증 회원가입 (Resend OTP + B1234 백도어)
- 로그인 / 로그아웃 / 회원탈퇴 (탈퇴 흔적 '탈퇴한 사용자' 유지)
- 상품 등록 / 수정 / 삭제 / 목록 / 상세
- 태그 필터 (팝니다 / 삽니다 / 나눕니다 / 같이삽시다)
- 완료 상품 숨기기/보기 토글
- 제목·닉네임 검색, 페이지네이션 (12개)
- 같이삽시다: 한 배에 타기/하차, 출항 팝업
- 댓글 (1depth)
- 마이페이지 (내 정보 + 내 상품)
- 쪽지 기능 (대화 목록 / 상세 / 전송, 읽음 처리)
- 모바일 반응형 네비게이션
- Open Graph 메타 태그
- 배포: Vercel(프론트) + Render(백엔드)
