1. 게시글을 쓸 때 금액을 입력할 수 있음 (숫자) - 게시글에서 금액을 확인 가능
2. 게시글을 쓸 때 금액 뒤에 '흥정 가능' 플래그가 붙을 수 있음 - 게시글에서 플래그 확인 가능 
3. '같이삽시다' 는 "한 배에 타기" 버튼이 있고,  "한 배에 타기" 버튼이 눌려지면 "배에서 하차하기" 버튼이 나타남. 인원이 전부 차면 게시글 작성자는 해당 게시글에서 "출항"이라는 버튼을 발견할 수 있음. 이 버튼을 누르면 참여 신청을 한 사람의 이메일이 보이는 (팝업창)이 뜨고, 그 아래 "메일 전송"버튼이 있음. "한 배에 타기"를 누른 사람들의 아이디(이메일)로 "메일"이 발송됨 (이건 "메일 발송" 버튼만 만들고 기능은 구현하지 않음.)
4. '같이삽시다' 는 현재 참여 인원을 볼 수 있음

# 회원가입 기능 개선 

1. 회원가입을 요청할 경우 validation을 하고 (도메인 확인, 이메일 중복 확인)
2. validation에 통과한 경우 인증 메일을 전송했다는 팝업창이 뜸. 
3. 그러나 실제로 메일은 보내지 않음 (안내 문구: 실제로 메일은 가지 않았습니다. 인증코드: 1234 ) - 왜냐면 이건 프로토타입이니까 충분해
4. 사용자가 1234를 입력하면 가입 api를 태워서 실제로 회원가입이 가능하게 해줘 

# 회원탈퇴 기능 추가 
1. 마이페이지 안에 회원탈퇴 버튼을 추가합니다. (작게, 가장 하단에)
2. 버튼을 누르면 "진짜로 날 떠날거야?" 알림이 뜹니다. 남기/떠나기 버튼이 뜹니다.
3. 떠나기 버튼을 누르면 회원 탈퇴가 진행되며 계정이 삭제됩니다. 
4. 계정은 삭제되었지만 계정이 작성한 게시글, 댓글은 전부 남아 있습니다.
